# UI/UX Design Documentation

## Visual Style

- Animated gradient background with shifting purple, blue, and pink tones
- Glassmorphism panels with subtle blur and soft shadows
- Rounded corners and roomy spacing for modern visuals
- High contrast text and iconography for readability

## Layout

- Login/Register card is centered and mobile-friendly
- Sidebar for user information, stats, and filters
- Main content area for task creation and task list
- Modal overlay for task editing

## Interactive Behavior

- Tab switching between Login and Register
- Hover effects on buttons and task cards
- Animated gradient and subtle particle lighting
- Responsive reflow for tablets and phones

## Accessibility

- Clear input labels and placeholders
- Large tap targets for buttons
- Status information visible in the sidebar
