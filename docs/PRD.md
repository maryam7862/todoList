# Product Requirements Document

## Product Vision

TaskFlow is a secure authentication-based to-do application that helps users manage daily tasks with a modern animated interface and backend persistence.

## Primary Goals

- Support user registration and login
- Persist tasks to MongoDB per user
- Allow task creation, editing, completion, filtering, and deletion
- Provide task priority and due date metadata
- Display task statistics and personalized greetings
- Deliver a polished animated UI with responsive support

## Key Features

- User authentication using email and password
- JWT token authorization for secure API access
- Task CRUD operations
- Task filters: All, Active, Completed
- Priority labels: Low, Medium, High
- Task due date support
- Responsive layout for desktop and mobile

## Success Metrics

- Smooth login and task flow
- Fast API responses with stable persistence
- Easily readable task list and task organization
- Consistent data between frontend and backend
